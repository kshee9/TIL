1. 오늘은 GIT에 대한 개념과 사용법을 알아보는 강의를 들었다.

GIT : 프로젝트의 버전관리를 위한 도구 
SOURCETREE : GIT을 사용하기 쉽게 도와주는 도구 툴 

프로젝트 생성  > GIT 초기화(GIT이 관리하는 프로젝트로 만듬) > ADD(STAGING0 파일단위로 추가가능 > 커밋 (정보등록)

GIT 을 사용함으로써 프로젝트의 내용 변화를 알수있으며 
작업 내용, 어떤 대상이 작업을 했는지 , 커밋한 시간까지 알 수있어 변화를 알기 좋다.

REPO : REPOSITORY 저장소의 약자 !
원격 REPO : 내 컴퓨터의 프로젝트를 원격으로 등록하여 모든사람들이 볼수있게 함 .
로컬 REPO : 내 컴퓨터에만 있는 저장소를 의미함

원격 REPO >>>> 로컬 REPO 로 옮기는 것은 PULL 
로컬 >>>> 원격 으로 옮기는것은 PUSH 
로컬 컴퓨터가 없는 원격 REPO를 로컬로 옮기면 CLONE !!
